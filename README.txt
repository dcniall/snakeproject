Snaky Serenades

Author: Niall Deasy

Documentation for all classes can be found in the JavaDoc.zip - doc folder

To run the project
 
1. extract the binaries zip
2. open the extracted bin folder
3. navigate to the extracted bin folder in your console
4. run the following command once in that folder: java Main ensemble.csv 1
5. Result should be:
A maximal length 1-MRPS song for this 3-snake ensemble has 6 sings:
Rod, Jane, Rod, Jane, Freddy, Freddy.
